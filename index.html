Build export report feature — allow users to select format and download

Type Story
Assignee maya@company.com
Labels analytics export q2-2026
Component Reports

Description

As a user, I want to export report data in my preferred format (CSV, PDF, or Excel) so that I can analyse it outside the platform or share it with stakeholders.

The export flow should allow me to select the format, trigger the export job, and receive a download link or direct file download when the job is complete.

Acceptance Criteria

User can select export format from CSV, PDF, Excel
Export job is triggered on confirmation
Download begins automatically on job completion
Error state shown if export fails
Export button is disabled while a job is in progress
